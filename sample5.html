Search on Map App

Overview

  • This app provides an address search for user on map. User can search the desired address by provided input fields and choose a label for it, then the app returns the postal code related to the address and will bookmark the address on map.
  • Additionally the user has the option choose the address as favourite one, so the app will remember it and stores it in local storage. A button is provided if the user wants to clear the history which clears out the local storage.
  • Results are displayed in a stateful and informative manner with paging control.

    • Analysis

  • This app uses the API from http://geocoder.ca for retrieving postal code from address.
  • Additionally it uses following libraries to show the map and bookmarks.
  • Validation is done for input values for example only one checkbox is checked for being favourite address.
  • Clean coding was practiced in this work sample. All coding related to search address was defined in a class named myLib and an object of it was instantiated in the main scripts file and the provided functions in that class were called to do the needed functionality.
  • Local storage was used to keep the history data related to favorite places.
  • XMLHttpRequest is used to get data from API by providing address as follow
  • The return value from api would be postal code and latt and longt which are used for map marker.
