Knight vs dragon Game

Overview

  • This game is implemented by Java using GUI.
  • Object-oriented programming principles (like inheritance, aggregation and polymorphism) are used in this work sample.
  • Each player has a certain number of points at the start and the loser decided by who loses all of their health points first.
  • User can choose his/her knight character and weapon.
  • Some weapons will be available when user gets enough gold.
  • Knight and dragon switch their attack turn.
  • For each attack if the attack number is more than defense number, attacker wins and the difference will be decreased from the loser health points.

  • This work sample follow clean coding for separating the GUI and base object functionality.
