RESTful API in Node.js

Implemented endpoints in a RESTful API that supports full CRUD for the city table of the world sample database .

Overview:

REST ( REpresentational State Transfer) is designed to take advantage of existing protocols. It usually takes advantage of HTTP when used for Web APIs. Additionally, REST can return XML, JSON, YAML or any other format depending on what the client requests.

RESTful API constraints:

Client-Server, Stateless, Cache, Unifrm Interface, Layered System, Code on Demand

CRUD operation is implemented as follow.

C (Create) R (Read) U (Update) D (Delete)

Note:

Postman is used to test the API post request for ( Create, Update and Delete)
