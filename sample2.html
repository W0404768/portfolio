Zap-A-Zombie

This Game implemented in JavaScript

Overview

  • This app provides a game which gives the player the choice to select the board dimension by inputting the numbers of rows and columns.
  • There is a timer which count down and the player can keep playing until timer reaches zero.
  • Start button would be disabled during play to avoid any interruption until time is over.
  • The user can start game by clicking the start button and try zap a zombie which appears randomly in one of cells.
  • If player can catch the zombie by clicking on the cell which zombie is showing up, player can collect one score otherwise the zombie will disappear and will show up in another random cell until time is over.
  • Validation is implemented which let player choose game board only between 1 and 10 for rows and cells.

    • Analysis

  • Dynamic creating game board was done by using createElement and appendChild functionalities of JavaScript based on input values for rows and cells by player.
  • setInterval function was used for game timing.
  • User input validation was done like limiting number of game board and disabling start button during play.
  • The click event of board game cells was used to implement the game if zombie is clicked to collect score or not.
  • It was tricky that how we can track clicking event if player clicked the zombie, which the click event placed inside showZombie() function when the random number is selected for putting zombie on board the click event also is defined for the same index number, so if there is any click by player we can catch it by that event listener.
  • The random function was used to randomly appearing the zombie.
  • The background image was used to decorate the game board and showing zombie. It was a little challenging to put both photos at the same time as background image when zombie shows up but the result is satisfying as it shows the zombie behind the door decoration.
